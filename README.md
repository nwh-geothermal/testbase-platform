# 陕西省地热能开发利用技术中试基地宣传平台

## 项目简介

这是一个为"陕西省地热能开发利用技术中试基地"打造的现代化宣传平台，采用 React、TypeScript 和 Next.js 技术栈开发。平台旨在展示地热能全产业链技术集成与推广成果，体现可视化、实时化、示范化的特色。

## 功能特色

- 🏠 **首页展示** - 震撼的英雄区域和核心数据展示
- 🔬 **技术体系** - 完整的技术创新链条展示
- 📊 **实时监控** - 基于图表的数据可视化仪表板
- 🏆 **成果展示** - 专利、论文、获奖等科研成果
- 🤝 **产学研合作** - 合作伙伴和合作模式展示
- 📱 **响应式设计** - 适配各种设备屏幕
- ✨ **动画效果** - 流畅的页面交互体验

## 技术架构

- **前端框架**: Next.js 14 (App Router)
- **开发语言**: TypeScript
- **样式框架**: Tailwind CSS
- **动画库**: Framer Motion
- **图表库**: Recharts
- **图标库**: Lucide React
- **主题管理**: next-themes

## 快速开始

### 环境要求

- Node.js 18.17 或更高版本
- npm 或 yarn 包管理器

### 安装依赖

```bash
npm install
# 或
yarn install
```

### 启动开发服务器

```bash
npm run dev
# 或
yarn dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看项目。

### 构建生产版本

```bash
npm run build
# 或
yarn build
```

### 启动生产服务器

```bash
npm start
# 或
yarn start
```

## 核心功能模块

### 1. 技术创新链条

展示"基础研究—技术攻关—技术熟化—成果应用"的完整技术创新体系，包含：

- 各阶段详细介绍
- 关键成果展示
- 数字孪生技术演示

### 2. 实时监控仪表板

基于模拟数据的可视化监控系统，包含：

- 关键指标监控（取热功率、地热温度、流量、系统效率）
- 能源产出趋势图表
- 温度监控曲线
- 系统运行状态分析
- 智能预警系统

### 3. 成果展示

全面展示科研成果，包含：

- 专利技术统计
- 学术论文发表
- 人才培养成果
- 技术创新突破
- 示范项目案例
- 获奖荣誉展示

### 4. 产学研合作

展示合作生态系统，包含：

- 合作伙伴分类展示
- 合作模式介绍
- 合作成果统计
- 联系方式信息

## 设计特色

### 视觉设计

- **配色方案**: 采用地热能主题色彩（橙色、蓝色、绿色）
- **渐变效果**: 多处使用渐变色增强视觉层次
- **卡片设计**: 现代化卡片布局，提升内容可读性

### 交互体验

- **滚动动画**: 基于 Framer Motion 的流畅滚动动画
- **悬停效果**: 卡片悬停阴影和变换效果
- **响应式**: 完美适配桌面、平板、手机各种设备

### 数据可视化

- **图表库**: 使用 Recharts 实现专业的数据图表
- **实时感**: 模拟实时数据更新效果
- **交互式**: 图表支持悬停提示和交互

## 开发说明

### 样式系统

项目使用 Tailwind CSS 作为主要样式框架，配合自定义 CSS 变量实现主题色彩系统。

### 组件设计

- 组件采用函数式设计，使用 TypeScript 进行类型定义
- 使用 "use client" 指令标识客户端组件
- 动画组件基于 Framer Motion 实现

### 数据管理

- 当前使用模拟数据展示功能
- 为后续接入真实 API 预留接口
- 图表数据支持动态更新
